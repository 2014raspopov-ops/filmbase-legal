Privacy Policy

FilmBase Video Uploader — desktop application for automated publishing of video content to TikTok.

Last updated: April 23, 2026

1. Overview

FilmBase Video Uploader does not collect, store, or process any personal data from TikTok users or any third parties. This application is a private internal tool used exclusively by FilmBase to publish marketing content.

2. Data We Do NOT Collect

This application does not collect:

  • Personal information of any users
  • TikTok user data
  • Browsing history or behavioral data
  • Location data
  • Any analytics or tracking data

3. TikTok API Access

The application uses TikTok's Content Posting API exclusively to upload video files to the FilmBase TikTok business account. The application only accesses its own account credentials (access tokens) stored locally on the operator's machine. No TikTok user data is accessed or processed.

4. Data Storage

The only data stored locally is the TikTok API access token for the FilmBase account, stored on the operator's local machine. This token is never transmitted to any third parties.

5. Third-Party Services

This application interacts solely with TikTok's official API endpoints. No other third-party data services are used.

6. Changes to This Policy

If we make changes to this Privacy Policy, we will update the "Last updated" date at the top of this page.
